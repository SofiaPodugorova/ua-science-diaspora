
# Афіліації українських дослідників-економістів в OpenAlex, 2018–2024
Невеликий пайплайн, який досліджує, як змінювалися інституційні афіліації українських дослідників в Economics за період 2018–2024, на основі бібліографічної бази даних OpenAlex.

## Дослідницькі питання
1. Як змінювалася частка українських афіліацій серед когорти українських дослідників?
2. Де публікувалися автори з українською афіліацією після 2022 року?
3. Чи відрізняються патерни зміни афіліацій між дослідницькими полями?

## Стек
Python (requests, pandas, matplotlib), SQLite, Jupyter.
## Як відтворити
Потрібно: Python 3.10+, git.

    git clone https://github.com/SofiaPodugorova/ua-science-diaspora.git
    cd ua-science-diaspora
    python -m venv .venv
    .venv\Scripts\Activate.ps1
    pip install -r requirements.txt
    python src/fetch.py
    python src/parse.py
    jupyter lab

Далі в JupyterLab обов'язково по порядку:
1. Відкрити notebooks/01_cleaning.ipynb → Run → Run All Cells. Дочекатись, поки виконається остання клітинка (вона створює файл panel).
2. Тільки після цього відкрити notebooks/02_analysis.ipynb → Run → Run All Cells.
Якщо запустити 02 раніше за 01, буде FileNotFoundError — другий ноутбук читає файл, який створює перший.

## Структура
- src/fetch.py — клієнт OpenAlex API з пагінацією
- src/parse.py — нормалізація JSON → SQLite
- notebooks/01_cleaning.ipynb — профілювання даних, дедуплікація, побудова панелі
- notebooks/02_analysis.ipynb — три дослідницькі питання, п'ять графіків
- data/clean/author_year_country.parquet — фінальна аналітична панель (створюється у 01)

## Основні висновки (вибірка з 5,000 робіт у концепті Economics)
- Q1. Частка українських афіліацій трималася на рівні 0.86–0.91 у 2018–2022 роках, а потім впала до 0.80 у 2023–2024. Узгоджується з типовим лагом публікацій ~1 рік.
- Q2. Топ напрямків (не-Україна) у 2022–2024: Польща (57), США (28), Словаччина (25), Велика Британія (20), Угорщина (19). Домінують сусіди та англомовні країни.
- Q3. Частки зміни афіліацій по полях варіюються від ~15% (Renewable energy) до ~1.5% (Medicine). Вищі показники у підполях зі сильнішими міжнародними policy/funding зв'язками. Розміри вибірок по полях невеликі — патерн орієнтовний.

## Обмеження
- Вибірка з 5,000 робіт з одного концепту OpenAlex; цифри ілюстративні, а не оцінки генеральної сукупності.
- Зміна афіліації ≠ фізичне перемещення.
- 2024 неповний — OpenAlex індексує роботи з лагом; падіння у 2024 відображає як міграцію, так і неповну індексацію.
- Концепти OpenAlex deprecated; Topics дав би точнішу класифікацію.
- Дедуплікація авторів спирається на author_id OpenAlex і незалежно не верифікована.
