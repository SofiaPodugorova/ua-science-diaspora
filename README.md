\# Афіліації українських дослідників в OpenAlex, 2018–2024



Невеликий пайплайн обробки даних, який досліджує, як змінювалися інституційні афіліації українських дослідників у період 2018–2024, на основі бібліографічної бази даних OpenAlex.



\## Дослідницькі питання



1\. Як змінювалася частка українських афіліацій серед когорти українських дослідників?

2\. Де публікувалися автори з українською афіліацією після 2022 року?

3\. Чи відрізняються патерни зміни афіліацій між дослідницькими полями?



\## Стек



Python (requests, pandas, matplotlib), SQLite, Jupyter.



\## Як відтворити



&#x20;   pip install -r requirements.txt

&#x20;   python src/fetch.py     # завантажує \~5,000 робіт у data/raw/ (\~1 хв)

&#x20;   python src/parse.py     # створює data/clean/works.db

&#x20;   jupyter lab             # потім запустити notebooks/ по порядку



\## Структура



\- `src/fetch.py` — клієнт OpenAlex API з пагінацією

\- `src/parse.py` — нормалізація JSON → SQLite

\- `notebooks/01\_cleaning.ipynb` — профілювання даних, дедуплікація, побудова панелі

\- `notebooks/02\_analysis.ipynb` — три дослідницькі питання, п'ять графіків

\- `data/clean/author\_year\_country.parquet` — фінальна аналітична панель (створюється у 01)



\## Основні висновки (вибірка з 5,000 робіт)



\- \*\*Q1.\*\* Частка українських афіліацій серед української когорти трималася на рівні 0.86–0.91 у 2018–2022 роках, а потім впала до 0.80 у 2023–2024 — узгоджується з типовим лагом публікацій \~1 рік після подій 2022 року.

\- \*\*Q2.\*\* Топ напрямків для авторів з українською афіліацією до 2022 у період 2022–2024: Польща (57 авторів), США (28), Словаччина (25), Велика Британія (20), Угорщина (19).

\- \*\*Q3.\*\* Частки зміни афіліацій по полях варіюються від \~15% (Renewable energy) до \~1.5% (Medicine), але розміри вибірки по полях невеликі, тому патерн є радше орієнтовним, ніж надійним.



\## Обмеження



\- Вибірка з 5,000 робіт з одного концепту OpenAlex; цифри ілюстративні, а не оцінки генеральної сукупності.

\- "Зміна афіліації" — це непрямий проксі для міграції. Подвійні афіліації та віддалена співпраця є поширеними.

\- Концепти OpenAlex застаріли; таксономія Topics дала б точнішу класифікацію в подальшій роботі.

\- Дедуплікація авторів спирається на вбудований `author\_id` OpenAlex і незалежно не верифікована проти ORCID або
